# Pokemon Tools — Ideias de ferramentas futuras

Não fazem parte do spec V1. Apenas anotações pra revisitar depois.

## Sinergia com Binder

### Pull Tracker
Registra abertura de boosters: pull rate, valor estimado, "card of the pack". Conversa com Binder pra adicionar pulls direto numa página.

### Trade Log
Histórico de trocas (deu X, recebeu Y, pra quem). Atualiza Binder automaticamente.

### Wishlist
Lista de cartas faltantes (puxa do Binder os "missing" + adições manuais). Útil em loja/feira.

## Sinergia com Battle Assistant

### Damage Calculator
Calculadora rápida: ataque base × weakness/resistance × bonuses. Pode virar modal dentro do Battle Assistant também.

### Deck Builder / Tracker
Monta decks (lista de cartas do TCG API), associa win rate por deck via Battle Assistant.

## Standalone

### Coin Flip / Dice
Utilitário rápido com histórico (já existe parcialmente no Battle Assistant, vira widget standalone também).
